使用TensorFlow去训练自己生成的验证码，

当前进度：

验证码自动生成---完成


直接运行 ：
python TensorFlow_cnn_train.py
则开始训练，当前成功率设定为50%则停止，
训练完毕确保训练后模型在同一文件夹下则运行
python TensorFlow_cnn_test_model.py
实时生成新验证码并且进行预测

当前程序设定一个成功率0.5或者0.7

我设置为当训练达到0.7 准确性就停止并且保留其模型。。

大概训练到 5k多次就能达到这个准确度。
